O que é jazz?


Jazz é um tipo de música que foi inventado nos Estados Unidos.
A música jazz combina a música afro-americana com a música europeia.
O jazz se tornou popular pela primeira vez na década de 1910.
Ainda é uma música popular para tocar e
ouvir por causa dos diferentes estilos.
